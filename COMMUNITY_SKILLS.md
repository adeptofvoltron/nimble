# Community Skills

A list of known community-contributed Nimble skills. Install any of them with:

```bash
nimble add <binding> <repository-url>
```

## Available skills

| Skill | Repository | Description |
|-------|------------|-------------|
| translate | https://github.com/adeptofvoltron/nimble-translate-skill | Translate selected text |
| explain | https://github.com/adeptofvoltron/nimble-explain-skill | Explain selected text or code |

---

To publish your own skill, build it following [.ai/skill-build.md](.ai/skill-build.md) and open a PR to add it here.
